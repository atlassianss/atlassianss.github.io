vsftpd搭建ftp服务器

Created2021-09-24|Updated2021-09-24
|Post View:

rpm -qa | grep vsftpd
yum install vsftpd -y
​

systemctl start vsftpd //启动服务
systemctl status vsftpd //开机自启
​

修改/etc/vsftpd/vsftpd.conf 配置文件
vim /etc/vsftpd/vsftpd.conf
local_root=/var/www/html
chroot_local_user=YES
anon_root=/var/www/html
//local_root 针对系统用户，anon_root 针对匿名用户 // /var/www/html 是你改的上传下载路径
配置 vsftpd 匿名用户上传的权限
vim /etc/vsftpd/vsftpd.conf

配置模板：
vsftpd.conf:
anonymous_enable=NO
local_enable=YES
write_enable=YES
local_umask=022
dirmessage_enable=YES
xferlog_enable=YES
xferlog_file=/var/log/vsftpd.log ##主动模式关闭
#port_enable=YES
#connect_from_port_20=YES
xferlog_std_format=YES
ascii_upload_enable=YES
ascii_download_enable=YES
chroot_list_enable=YES
chroot_list_file=/etc/vsftpd/chroot_list
listen=YES
pam_service_name=vsftpd
userlist_enable=YES
tcp_wrappers=YES
local_max_rate=500000
max_clients=200
max_per_ip=10 ##被动模式开启
pasv_enable=YES
pasv_min_port=3500
pasv_max_port=4000
​

user_list:

vsftpd userlist

If userlist_deny=NO, only allow users in this file

If userlist_deny=YES (default), never allow users in this file, and

do not even prompt for a password.

Note that the default vsftpd pam config also checks /etc/vsftpd/ftpusers

for users that are denied.

root
bin
daemon
adm
lp
sync
shutdown
halt
mail
news
uucp
operator
games
nobody
